# ai-presents-app-ios

iPhone-first app for:
- Upcoming birthdays (from iOS Contacts)
- Gift idea capture per contact
- Smart reminders (30/14/7/2 days)
- iCloud sync (CloudKit)
- AI-assisted gift ideation (OpenRouter), privacy-sparse by design

## Product Principles
1. Privacy-first, minimal data flow
2. Native Apple UX (SwiftUI, HIG-aligned)
3. Reliable reminders before novelty
4. AI as assistive layer, not required for core utility

## Scope v1
- iPhone only
- German UI first
- OpenRouter integration planned in MVP architecture
