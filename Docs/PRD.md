# PRD — ai-presents-app-ios (v1)

## Goal
Nutzer sieht kommende Geburtstage, pflegt Geschenkideen pro Kontakt und wird rechtzeitig erinnert.

## Core Jobs
- „Wessen Geburtstag ist bald?“
- „Was könnte ich schenken?“
- „Wann muss ich spätestens kaufen?“

## Must-have Features
1. Contacts Birthday Import (permission-based)
2. Birthday Timeline (Heute / 7 Tage / 30 Tage)
3. Gift Ideas per Contact
4. Reminder Engine (30/14/7/2)
5. iCloud Sync
6. German localization

## AI (OpenRouter) — v1 planned
- Idea generation from tags + budget + past gifts
- Similarity guard (avoid repeated gift types)
- Message drafting (birthday text)

## Non-functional
- Offline-capable core
- Fast launch (<1.5s warm)
- Deterministic reminders
- Privacy sparse data policy
